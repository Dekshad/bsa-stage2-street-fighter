# 🥊 Street Fighter JS clone

Сучасний клон легендарного файтингу **Street Fighter**, реалізований на чистому JavaScript (Vanilla JS) з використанням сучасних маніпуляцій з DOM та візуальних ефектів.

---

## ✨ Основні особливості та функціонал

*   **Динамічне прев'ю бійців**: При натисканні на картку бійця на екрані вибору з'являється панель з його детальними характеристиками (HP, ATK, DEF) та анімацією зображення.
*   **Бойова механіка**:
    *   **Звичайні атаки та захист**: Можливість наносити удари та блокувати їх. Під час блокування боєць повністю або частково ухиляється від шкоди залежно від сили блоку.
    *   **Заборона подвійних дій**: Персонаж не може наносити удар, якщо знаходиться в стані блокування.
*   **Критичні комбо-удари (Critical Combos)**:
    *   Можливість провести суперудар, натиснувши одночасно 3 клавіші.
    *   Суперудар завдає подвійну чисту шкоду (`2 * attack`), ігнорує будь-який блок супротивника та має 10-секундний час перезарядки (cooldown).
*   **Спливаючі числа шкоди (Floating Damage Numbers)**:
    *   Яскраві візуальні ефекти відображення шкоди під час бою.
    *   Жовті літаючі індикатори для звичайних ударів, великі червоні з анімацією обертання для критичних ударів та сірий напис `BLOCKED` при повному поглинанні удару.
*   **Вікно оголошення переможця**: Модальне вікно з ім'ям та анімованим зображенням переможця, яке плавно з'являється після K.O. та дозволяє перезапустити гру в один клік.
*   **Покриття юніт-тестами**: Основна математична логіка бою та взаємодія з сервісами покрита тестами на фреймворку **Vitest**.

## 🎮 Керування в бою (Controls)

| Дія | 🥋 Гравець 1 (Ліворуч) | 🥋 Гравець 2 (Праворуч) |
| :--- | :---: | :---: |
| **Атака (Attack)** | `A` | `J` |
| **Захист (Block)** | `D` | `L` |
| **Суперудар (Critical Combo)** | `Q` + `W` + `E` | `U` + `I` + `O` |

## 🏃‍♂️ Швидкий старт (Setup & Start)

1.  **Встановлення залежностей та гіт-хуків**:
    ```bash
    npm run setup
    ```

2.  **Запуск локального сервера розробки (Vite)**:
    ```bash
    npm run dev
    ```

3.  **Перегляд у браузері**:
    Відкрийте посилання [http://localhost:7800/](http://localhost:7800/)

## 🧪 Тестування та перевірка коду (Tests & Linting)

Проєкт налаштований відповідно до найсуворіших стандартів Clean Code, ESLint (Airbnb) та Prettier.

*   **Запуск юніт-тестів (Vitest)**:
    ```bash
    npm run test
    ```
*   **Запуск тестів у режимі спостереження**:
    ```bash
    npm run test:watch
    ```
*   **Перевірка коду лінтером (ESLint)**:
    ```bash
    npm run lint
    ```
*   **Автоматичне виправлення стилістичних помилок**:
    ```bash
    npm run lint:fix
    ```
